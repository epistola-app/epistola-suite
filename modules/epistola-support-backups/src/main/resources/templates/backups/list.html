Backups

A backup is a point-in-time copy of all of this tenant's published catalog content, stored on the Epistola hub — unpublished drafts are not included. Daily backups run automatically; you can also back up on demand. Restoring is destructive — it replaces every catalog in this tenant with the snapshot's contents (any unpublished drafts are lost).
